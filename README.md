# STO DUA

Clean base project for building a new website with Next.js.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## If Changes Are Not Visible

If you changed code but still see old styles/layout:

```bash
# stop running dev server first (Ctrl + C in terminal)
npm run dev:clean
```

Then open exactly `http://localhost:3000` and hard refresh the page (`Cmd + Shift + R` on macOS).

## Build

```bash
npm run build
npm start
```
