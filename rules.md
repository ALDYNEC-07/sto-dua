# Правила работы

1. После каждого нового изменения обязательно добавлять комментарий простым и понятным языком: что изменено и зачем.
2. Мобильная версия всегда в приоритете: сначала проверяем и настраиваем поведение для мобильных экранов, затем для десктопа.
3. После каждого изменения обязательно предлагать 1 идею для улучшения текущего изменения.
